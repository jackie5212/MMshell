# MMShell

MMShell 是一个基于 `Tauri + Rust` 的新一代终端与远程运维客户端，目标是达到并超越同类主流产品的体验与稳定性。

## 当前阶段

- 年度计划：Y1（第1年）
- 执行窗口：M1-M2（前两个月）
- 目标：打通 SSH 主链路，建立多会话与稳定性基础

## 本目录已生成内容

- `docs/PRD-v1-M1-M2.md`：前两个月产品需求定义
- `docs/技术架构-v1-M1-M2.md`：架构设计与模块划分
- `docs/第一年前两个月-按天执行计划.md`：Day001-Day060 每日执行表
- `docs/M1-M2-里程碑与验收标准.md`：月度里程碑与验收门槛
- `docs/M1-M2-风险与应急预案.md`：风险清单、触发条件与应急动作
- `docs/M1-M2-Jira导入模板.csv`：任务导入模板（CSV）

## 执行原则

- 每天必须有交付（代码、文档、测试、复盘至少一项）
- 每周必须有可演示版本
- 每月必须有里程碑验收
- 发布必须满足质量门禁，不达标不发布
