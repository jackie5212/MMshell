# MMShell M1 用户说明（Day029）

## 1. 启动方式

在项目目录执行：

```powershell
npm.cmd run tauri dev
```

## 2. 功能说明

### 2.1 会话管理

- 在表单输入 `Session name`、`Host`、`Port`、`Username`
- 点击 `Save Session` 保存会话
- 已保存会话会展示在 `Saved Sessions` 列表

### 2.2 连接测试

- 点击某条会话的 `Reachability Test`
- 系统会进行目标地址连通性检测
- 状态会显示为 `Connecting`、`Connected` 或 `Failed`

### 2.3 标签页

- 点击 `Open Tab` 打开该会话对应的标签
- 标签支持切换与关闭
- 切换标签后内容保持不丢失

### 2.4 复制粘贴

- 在终端占位区输入文本
- 点击 `Copy` 复制当前标签内容
- 点击 `Paste` 将剪贴板内容追加到当前标签

### 2.5 活动日志

- 页面底部 `Recent Activity Logs` 展示最近操作日志
- 用于排查保存会话、读取会话、连接测试等动作

## 3. 已知限制

- 当前连接测试仅验证端口可达，不代表 SSH 认证通过
- 终端占位区尚未接入真实 SSH 命令流
