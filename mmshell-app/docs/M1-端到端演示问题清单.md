# M1 第一次端到端演示问题清单（Day024-Day025）

## P0/P1 问题处理记录

1. **P1：Windows PowerShell 无法直接执行 npm**
   - 现象：`npm.ps1` 被执行策略阻断
   - 修复：`launch.json` 使用 `npm.cmd run tauri dev`
   - 状态：已修复

2. **P1：引入 ssh2 依赖后 Windows 缺少 perl 导致编译失败**
   - 现象：`openssl-sys` 构建失败，提示 `Command 'perl' not found`
   - 修复：移除 `ssh2`，暂时改为 TCP 连通性测试
   - 状态：已修复（后续再接入纯 Rust SSH 库）

3. **P1：早期页面功能弱、无可视化状态**
   - 现象：保存会话后缺少连接过程反馈
   - 修复：增加连接状态显示（connecting/connected/failed）
   - 状态：已修复
