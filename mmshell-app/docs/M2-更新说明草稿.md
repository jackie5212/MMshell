# M2 更新说明草稿（Day058）

## 新增能力

- 会话分组字段（`group`）支持保存和展示。
- 会话搜索支持名称、主机、用户名、分组。
- 会话 JSON 导入导出（支持合并更新）。
- 连接策略可配置：超时、重试次数、重试退避、严格主机校验。
- 主机指纹预览与信任列表（known_hosts 基础流程）。
- 诊断快照导出（会话数量、日志、基础指标）。

## 稳定性改进

- 连接状态支持自动清理过期状态，避免死连接残留。
- 增加运行时指标统计（成功/失败/延迟/重试）。
- 增加高频输入节流提交，缓解快速输入掉字现象。

## 用户体验

- 会话列表支持分组前缀展示。
- 错误提示改为用户可读文案，减少原始报错暴露。
- 增加批量测试入口和快捷键操作。
