# M2 稳定性跑测说明（Day055-Day056）

## 跑测脚本

- 脚本路径：`scripts/stability-run.ps1`
- 默认参数：8小时、20秒心跳一次

## 执行方式

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\stability-run.ps1 -DurationHours 8 -IntervalSeconds 20
```

## 输出

- 输出日志：`docs/M2-稳定性跑测日志.txt`
- 记录内容：每次心跳时间戳、开始与结束时间

## 处理策略

- 若出现 UI 卡死/连接状态不更新：优先排查运行时状态清理逻辑
- 若出现连接大量失败：优先排查网络、超时和重试参数
